# Крамаруха Владислав
## Тестовое задание для MediaSoft

Тестовый проект "Список задач"
Реализованная функциональность:
- Создание / редактирование / удаление / восстановление задач
- Фильтрация / сортировка списков задач

## Установить node модули

```sh
npm install
```

### Запустить проект

```sh
npm run dev
```
